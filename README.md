# Knowledge Injection in LLMs 📚

Bienvenue dans mon dépôt GitHub consacré aux présentations et résumés de papiers scientifiques récents sur :

- **Knowledge Injection dans les Large Language Models (LLMs)**
- **Retrieval-Augmented Generation (RAG)**
- **Gestion des conflits de connaissances**
- **Graphes de connaissances** (Neo4j, UMLS, ConceptNet)

> ⚠️ Ces présentations ne sont **pas des travaux originaux des auteurs des papiers**.  
> Elles représentent mes propres résumés et analyses dans le cadre de mon parcours de doctorat en France.  
> Je m'excuse auprès des auteurs originaux si une erreur ou une simplification se glisse dans mes présentations.  

Ce dépôt est **open source** : vous pouvez utiliser mes présentations comme vous le souhaitez, **sans restriction**.
